# 폴더 선택 기능 개선

## 문제점
1. PowerShell 명령 타임아웃 (30초)
2. 서버 프로세스에서 GUI 대화상자가 나타나지 않음
3. Chat 기능에서도 동일한 에러 발생

## 해결 방법

### 1. 별도 Python 스크립트 사용
- `scripts/select_folder.py`: tkinter를 사용한 폴더 선택
- 별도 프로세스로 실행하여 GUI가 제대로 나타나도록
- CREATE_NEW_CONSOLE 플래그로 새 창에서 실행

### 2. 타임아웃 증가
- 30초 → 120초 (2분)로 증가
- 사용자가 충분히 선택할 시간 확보

### 3. 에러 처리 개선
- 명확한 에러 메시지
- 대체 방법 안내 (경로 직접 입력)
- UI에서 로딩 상태 표시

### 4. 폴백 메커니즘
1. 별도 스크립트 실행 (우선)
2. 직접 tkinter 사용 (백업)
3. 에러 시 사용자에게 경로 직접 입력 안내

## 사용 방법

### 폴더 선택 버튼 클릭 시
1. "폴더 선택 대화상자를 여는 중..." 메시지 표시
2. 새 창에서 폴더 선택 대화상자 열림
3. 선택한 경로가 자동으로 입력란에 추가
4. 실패 시 명확한 안내 메시지

### 대체 방법
- 경로를 직접 입력란에 입력
- 파일 탐색기에서 경로 복사 후 붙여넣기
- 세미콜론(;)으로 여러 경로 구분

## 기술적 세부사항

### 서버 측 (`app/server.py`)
- `subprocess.CREATE_NEW_CONSOLE`: 새 콘솔 창에서 실행
- 타임아웃: 120초
- 에러 처리: 명확한 메시지 반환

### 클라이언트 측 (`app/ui/index.html`)
- AbortSignal.timeout: 2분 타임아웃
- 로딩 상태 표시
- 에러 시 사용자 안내

## 문제 해결

### 여전히 대화상자가 나타나지 않을 때
1. Python에 tkinter가 설치되어 있는지 확인
2. 서버를 재시작
3. 경로를 직접 입력 사용

### 타임아웃이 계속 발생할 때
- 네트워크 지연 가능성
- 서버 부하 확인
- 경로 직접 입력 권장
